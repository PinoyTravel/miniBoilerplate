MiniBoilerplate

Download v1.4 Github Examples

Description

Looking to create a new jQuery plugin in CoffeeScript using Test-driven development? If so, MiniBoilerplate is what you've been looking for.

As an example, the MiniBoilerplate has been used to create the entire suite of MiniJs plugins.

Features

  • jQuery Plugin Default

    Includes default JavaScript, HTML, CSS and Readme file templates

  • Test-driven Development

    A Jasmine test suite and helper default templates provided and Travis-CI compatible

  • CoffeeScript

    JavaScript code is written in CoffeeScript

  • Libraries

    jQuery, Jasmine, Jasmine jQuery and SinonJs libraries included

  • Workflow

    Instant coffeescript compilation and test suite status

  • Easy to install

    It takes less than 5 minutes to get started

Requirements

Getting Started

  • Clone repository

    git clone git://github.com/miniJs/miniBoilerplate.git
cd miniBoilerplate

  • Install needed libraries

    bundle install

  • Compile CoffeeScript

    bundle exec guard

  • Run test suite in the terminal (headless)

    bundle exec guard

  • Run test suite in the browser

    bundle exec rake jasmine
open http://localhost:8888/

  • Run test suite remotely using Travis-CI

    1. Hook up your Githup repository to Travis.
    2. Update your repository name at the very top of the README file (four spots).
    3. Push your local Git repository to Github and go to Travis to check the buld status.
    4. The README file of your Github repository should now display the latest build status.

  • Plugin usage

    $('#element').pluginName({
  message: 'hi',
  callback: function() {
    alert('bye');
  }
});
