MiniBoilerplate

Download v0.1 Github Examples

Description

Looking to create a jQuery plugin in CoffeeScript using Test-driven development? If so, MiniBoilerplate is what you've been looking for.

As an example, MiniBoilerplate has been used to create the entire suite of MiniJs plugins.

Features

  • jQuery Plugin Default

    Includes default JavaScript, HTML, CSS and Readme file templates

  • Test-driven Development

    Jasmine test suite, default helper templates provided and Travis-CI compatible

  • CoffeeScript

    JavaScript code is written in CoffeeScript

  • Libraries

    jQuery, Jasmine and Jasmine jQuery libraries included

  • Workflow

    Instant CoffeeScript compilation and test suite status

  • Easy to install

    It takes less than 5 minutes to get started

Requirements

  • Bundler gembundler.com

    gem install bundler

  • Homebrew mxcl.github.com/homebrew

    ruby <(curl -fsSkL raw.github.com/mxcl/homebrew/go)

  • Qt qt.nokia.com

Getting Started

  • Download MiniBoilerplate and go to project folder

    cd path/to/miniboilerplate

  • Install needed libraries

    bundle install

  • Compile CoffeeScript

    bundle exec guard

  • Run test suite in the terminal (headless)

    bundle exec guard

  • Run test suite in the browser

    bundle exec rake jasmine
open http://localhost:8888/

  • Run test suite remotely using Travis-CI

    1. Hook up your Githup repository to Travis
    2. Update your repository name at the very top of the README file (four spots)
    3. Push your local Git repository to Github and go to Travis to check the build status
    4. The README file of your Github repository should now display the latest build status

  • Plugin usage

    $('#element').pluginName({
  message: 'hi',
  callback: function() {
    alert('bye');
  }
});
