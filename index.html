Description

Looking to create a new jQuery plugin in CoffeeScript using Test-driven development? If so, miniBoilerplate is your solution.

As an example, the boilerplate has been used to create every minijs.com plugins.

Test-driven development

With the boilerplate, it's easy to go for Test-driven development while creating your plugin with Jasmine.

To be able to run the test suite, you'll need to install the Jasmine ruby gem. If you have bundler installed just follow these instructions:

cd path/to/plugin/directory
bundle install
rake jasmine
open http://localhost:8888/

Guard automatically tests your Jasmine specs when files are modified which will significantly increase your productivity. To do so you'll need to install phantomjs and follow these instructions.

cd path/to/plugin/directory
guard
